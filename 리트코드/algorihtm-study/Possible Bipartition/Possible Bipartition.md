# Review
- DFS + 빈출 문제
- 보통 Set으로 풀려하는데, 많이 지저분해지는 풀이

- 그룹이 배정이 되면, 그 그룹의 요소들은 더이상 변하지 않음
  - 각 그룹으로 쪼개진 요소들은 같은 그룹에 속하지 않는 요소들이기 때문
 
## 풀이
- 0 : 그룹에 배정이 안 된 요소들
- 1 : 그룹 A
- - 1: 그룹 B
  - "나(g)를 제외한 다른 값들은 -g에 할당"
