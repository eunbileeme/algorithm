> ex. 예제 3
- "aaadbbcc" // {a : 3}, {d : 1}, {b : 2}, {c : 2} -> 내림차순으로 정렬이 완료되어 있는 key-value => 내림차순 정렬 큐(우선순위 큐)
- **"a"부터 처리**
  - a // {a : 2}
    - "k만큼 기다려야" 다시 a를 줄 수 있음 -> 큐
    - 큐의 크기 = 얼마만큼 기다렸는지 = 그만큼 넣어야 다시 뺄 수 있으므로
      - **a = 2라는 엔트리를 따로 큐(일반 큐)에 격리**
     
### 핵심
- 어딘가에 대기를 시켜야 하는데, 어디에 보관시키면 될까?
