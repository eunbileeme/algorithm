## unexpected type : 원하는 타입(required)이 아니라 잘못된 타입(found)을 기재시 발생 ▶ 에러 체크된 부분의 타입을 required에서 나타난 타입으로 변경
```
ERROR unexpected type

required : value(요구하는 타입)

found : class(소스상 써준 타입)
```
